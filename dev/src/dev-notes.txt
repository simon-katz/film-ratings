Docker Postgres
===============

docker run -p 5432:5432 -e POSTGRES_USER=filmuser -e POSTGRES_DB=filmdb -e POSTGRES_PASSWORD=password postgres:alpine

psql -h localhost -p 5432 -U filmuser filmdb
